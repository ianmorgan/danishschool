What the site does

This site provides the same basic content as the old site. But this content is now stored within a database which can be maintained by authorised users through a simple set of admin screens. This includes

  • Adding and changing details for teachers and committee members
  • Creating news items
  • Creating new events (such as lessons) in the calendar
  • Uploading new files and images
  • Altering site content, e.g changing the home page
  • Adding lesson notes, handouts etc

Feel free to play around. You can logon with a test account of admin@danishschool.com with a password of "password". Just beware that I will occasionally rebuild the database with default content and hence any changes made will be discarded.

Technical Bits

For those who are interested, this site has been developed using Ruby on Rails. Briefly, this is a framework that makes it easier to write web sites. It is used by a number of larger companies, including Lonely Planet and www.domain.com.au. And appropriately, it's developed by a Dane

The demo site is deployed to a Commercial web host I've been using for a year or so and it's been reliable with good support. It is a basic developer account intended for use by indidividuals or small groups. This should be more thann sufficent for his web site given the small user base.
